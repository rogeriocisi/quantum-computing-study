# Quantum Computing Study: Roadmap & Implementation 🌌

This repository contains a comprehensive **12-month study roadmap** and practical implementations for transitioning from classical software engineering to **Quantum Computing Developer**, specifically targeting the **IBM Certified Associate Developer - Quantum Computation using Qiskit v2.X**.

## 🚀 The Mission
To transform classical programmers into quantum-ready developers by bridging the gap between **Linear Algebra** and **Quantum Hardware/Software** implementation using the latest **Qiskit v2.x** framework.

---

## 📅 12-Month Roadmap Summary

### Phase 1: Mathematical Foundations (Months 1-3)
*   **Month 1**: Linear Algebra (Hilbert Spaces, Dirac Notation, Complex Vectors).
*   **Month 2**: The Qubit & Bloch Sphere (Single-qubit gates: X, Y, Z, H).
*   **Month 3**: Entanglement (Multi-qubit systems, CNOT, Bell States).

### Phase 2: Fundamental Algorithms (Months 4-6)
*   **Month 4**: Oracles (Deutsch-Jozsa, Bernstein-Vazirani).
*   **Month 5**: Grover's Search (Amplitude Amplification).
*   **Month 6**: Shor's Algorithm (Quantum Fourier Transform - QFT).

### Phase 3: Industry Applications (Months 7-9)
*   **Month 7**: Quantum Chemistry (VQE - Variational Quantum Eigensolver).
*   **Month 8**: Quantum Machine Learning (VQC, Quantum Kernels).
*   **Month 9**: Optimization (QAOA - Quantum Approximate Optimization Algorithm).

### Phase 4: Hardware & Certification (Months 10-12)
*   **Month 10**: Noise & Error Mitigation (ZNE, M3, TREX).
*   **Month 11**: Portfolio Development (Complex project deployment).
*   **Month 12**: IBM Certification Prep (C1000-179).

## ⚙️ Setup Instructions

1.  **Create a Virtual Environment**:
    ```powershell
    python -m venv quantum_env
    .\quantum_env\Scripts\activate
    ```

2.  **Install Dependencies**:
    ```powershell
    pip install -r requirements.txt
    ```

3.  **Run your first circuit**:
    ```powershell
    python src/bell_state.py
    ```
