# Project Landing Page

This directory contains a simple web visualization of the **Quantum Computing Study Roadmap**.

## Contents
- `index.html`: Main structure of the landing page.
- `style.css`: Modern styling for the roadmap visualization.
- `hero.png`: Visual asset for the landing page.

## How to View
To view the progress dashboard locally, simply open the `index.html` file in any modern web browser.

## Purpose
The landing page serves as a visual summary of the 12-month roadmap and key milestones achieved during the study.
