# Study Progress Dashboard

This directory contains the visual interface for tracking the **Quantum Computing Study Roadmap**.

## Contents
- `index.html`: Main structure of the dashboard.
- `style.css`: Modern styling for the roadmap visualization.
- `hero.png`: Visual asset for the interface.

## How to View
To view the progress dashboard locally, simply open the `index.html` file in any modern web browser.

## Purpose
The dashboard serves as a visual summary of the 12-month roadmap and key milestones achieved during the study, providing a more interactive experience than the standard markdown files.
